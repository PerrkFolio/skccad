Crypto Currency Exchange Rates

This is a simple application for tracking exchange rates on p2p exchanges.

You can go to the page with real-time rates, or use the API:


GET: /p2p/crypto/fiat/volume/good_deals/ (is under development)
  • {crypto} - the cryptocurrency | type: string
  • {fiat} - the fiat currency | type: string
  • {volume} - the amount of cryptocurrency | type: int
  • {good_deals} - percentage of successful transactions | type: int (<100)
